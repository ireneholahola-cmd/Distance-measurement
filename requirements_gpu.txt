# GPU install notes
# 1. Install a CUDA-enabled PyTorch build that matches the target machine's Python
#    version and local CUDA runtime first. Example for CUDA 12.1:
#    pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121
# 2. Then install the remaining project dependencies:
#    pip install -r requirements_gpu.txt
#
# torch/torchvision are intentionally not pinned here because different teammates may
# use different CUDA versions. This avoids one machine's setup breaking another's.

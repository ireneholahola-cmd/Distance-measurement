# GPU 安装说明
# 1. 首先安装支持 CUDA 的 PyTorch 版本，该版本需与目标机器的 Python 版本
#    和本地 CUDA 运行时版本匹配。以 CUDA 12.1 为例：
#    pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121
# 2. 然后安装项目剩余依赖：
#    pip install -r requirements_gpu.txt
#
# 不固定 torch/torchvision 的版本，这样可以避免一台机器的环境配置导致其他成员的环境无法运行。

-r requirements_common.txt
